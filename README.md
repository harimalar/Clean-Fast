# ClearFast

Premium fasting tracker with free + pro model.

## Public legal pages

If GitHub Pages is enabled from `/docs`, these pages will be available at:

- `https://harimalar.github.io/Clean-Fast/`
- `https://harimalar.github.io/Clean-Fast/privacy.html`
- `https://harimalar.github.io/Clean-Fast/terms.html`
- `https://harimalar.github.io/Clean-Fast/medical-disclaimer.html`

## Why these files exist

This repo includes:

- External legal site pages for App Store compliance and user trust.
- Matching in-app legal text file for consistency between app and web policy.

## App-side legal sync

Use `app/AppLegalContent.swift` as your single source text for in-app legal cards/screens and external URL links.

## GitHub Pages setup

1. Open repository settings on GitHub.
2. Go to **Pages**.
3. Set **Source** to branch `main` and folder `/docs`.
4. Save and wait for deployment.
